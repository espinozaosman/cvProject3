Parse JSON in Bash with node-jq

July 3rd, 2021

If you retrieve JSON format data in Bash, and want to get the specific value, you can use node-jq to parse the JSON.


Example of JSON for variable STACK_INFO


1 # [ 2 # { 3 # OutputValue: 'Git information', 4 # } 5 # ]

Firstly, install node-jq:


1 npm install node-jq

then you can use jg like


1. /node_modules/node-jq/bin/jq

Finally, you can retrieve OutputValue in variable STACK_INFO as following:


1 STACK_COMMIT_ID=$(echo $STACK_INFO | ./node_modules/node-jq/bin/jq -r '.[0].OutputValue'); 2 commit id in $stackName stack: ${STACK_COMMIT_ID}"
